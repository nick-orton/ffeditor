# Project 

```
name: ffeditor
```

# Goal

A command-line tool for managing a collection of digital music files.

---

# Functionality

## TUI Interface

A terminal user interface (TUI) that can browse the filesystem and accept
commands via a command bar.

## Audio Conversion

- Convert `.opus` and `.m4a` files to `.mp3`
- Bulk convert an entire directory at once

## ID3 Tag Editing

- Edit ID3 tags on `.mp3` files
- Bulk tag multiple files at once (set shared fields like artist or album
  across a selection)
